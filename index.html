Интерактивный гайд по командной строке Windows (CMD и PowerShell)

Этот гайд поможет освоить базовые команды для работы с файлами, пользователями и процессами.

📂 Работа с файлами и папками

Основные команды

dir (CMD) / Get-ChildItem (PowerShell)
Показывает список файлов и папок.
dir
Get-ChildItem
cd (CMD) / Set-Location (PowerShell)
Переход в другой каталог.
cd C:\Users
Set-Location C:\Users
copy (CMD) / Copy-Item (PowerShell)
Копирование файлов.
copy file.txt D:\Backup\
Copy-Item file.txt -Destination D:\Backup\
del (CMD) / Remove-Item (PowerShell)
Удаление файлов.
del old.txt
Remove-Item old.txt

👥 Управление пользователями

Основные команды

net user (CMD)
Управление учетными записями.
net user Student123 MyPassword /add
Get-LocalUser (PowerShell)
Просмотр локальных пользователей.
Get-LocalUser
Remove-LocalUser (PowerShell)
Удаление пользователя.
Remove-LocalUser -Name Student123

⚙️ Мониторинг процессов и системы

Основные команды

tasklist (CMD)
Список запущенных процессов.
tasklist
taskkill (CMD)
Завершение процесса.
taskkill /IM notepad.exe /F
Get-Process (PowerShell)
Просмотр процессов.
Get-Process
Stop-Process (PowerShell)
Завершение процесса.
Stop-Process -Name notepad

📝 Практические задания

1. Создайте папку Practice на диске C. Скопируйте туда текстовый файл и удалите его через командную строку.
2. Создайте пользователя TestUser, проверьте список пользователей и удалите его.
3. Запустите notepad, найдите процесс и завершите его через команду.

Формат реализации: PDF, HTML или сайт. Данный пример — HTML-версия.
