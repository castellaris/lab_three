📂 Работа с файлами и папками

Рисунок 1 — команда dir

Команда dir выводит список файлов и папок в текущем каталоге.

Рисунок 2 — команда Get-ChildItem

Команда Get-ChildItem в PowerShell показывает содержимое каталога в структурированном виде.

Рисунок 3 — команда cd

Команда cd изменила рабочую директорию.

Рисунок 4 — команда Set-Location

Команда Set-Location выполняет аналогичный переход в PowerShell.

👥 Управление пользователями

Рисунок 5 — команда net user

Команда net user создаёт нового пользователя.

Рисунок 6 — команда Get-LocalUser

Команда Get-LocalUser выводит список локальных пользователей.

⚙️ Мониторинг процессов и системы

Рисунок 7 — команда tasklist

Команда tasklist показывает список всех запущенных процессов.

Рисунок 8 — команда taskkill

Команда taskkill завершает процесс по его PID.

Рисунок 9 — команда Get-Process

Команда Get-Process выводит список процессов с указанием дескрипторов, памяти и CPU.

📝 Практические задания

1. Создайте папку Practice на диске C. Скопируйте туда текстовый файл и удалите его через командную строку.
2. Создайте пользователя TestUser, проверьте список пользователей и удалите его.
3. Запустите notepad, найдите процесс и завершите его через команду.
4. Используя PowerShell, получите список всех процессов и найдите процесс браузера. Завершите его с помощью команды Stop-Process.
5. В командной строке создайте новый текстовый файл info.txt, добавьте в него строку «Hello CMD», затем отобразите его содержимое с помощью команды type.
